Hi, I'm Charles Chaverot,
Welcome to my portfolio.

Better coding in the 21st century means finding original and efficients ways to process increasing amounts of data and solve complex problems. I realized I loved coding through University and have been working on my skills ever since.

If you scroll down you will find my personal projects where I try to build and showcase my skills. Hope you like what you see!

Project 1: Call 4 Short (C4S)

C4S is a ChatGPT-based earnings call summarizer. It collects earnings call on-demand through the SeekingAlpha API, feeds it to ChatGPT and creates an interactive Chatbot where the user can interact with an augmented version of the AI.

Open project

Project 2: Crystal Credit

The goal of this project is to simulate a commercial bank's analysis of client profiles to identify common trends in credit card defaulters and use this information for screening of new applications. I am interested in improving skills of basic feature engineering and creating a machine learning model to process clients' profile information on a large scale.

Open project (not working rn)
